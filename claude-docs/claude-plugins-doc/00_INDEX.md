## 문서 목적
이 문서 세트는 Claude Code의 Plugin 시스템을 실무 관점에서 빠르게 이해하고, 직접 플러그인을 설계·작성·검증·배포할 수 있도록 구성한 분리형 문서다.

## 핵심 범위
- 플러그인과 standalone `.claude/` 구성의 차이
- 플러그인 기본 구조와 매니페스트
- skills, agents, hooks, MCP, LSP, settings, bin 구성 방식
- 설치 스코프와 CLI 관리 명령
- 테스트, 디버깅, 검증 절차
- 마켓플레이스 배포와 기존 설정 마이그레이션
