# Core Tools Reference
문서 ID: CC-TOOLS-CORE
버전: v1.0.0

## 1. Read
### 목적
파일 내용을 읽는다.
### 특징
- 무권한
- 코드 이해의 기본 도구

## 2. Glob
### 목적
패턴으로 파일을 찾는다.
### 특징
- 무권한
- 파일 구조 탐색에 적합

## 3. Grep
### 목적
파일 내용 패턴을 검색한다.
### 특징
- 무권한
- 특정 함수/문자열/에러 경로 탐색에 적합

## 4. Edit
### 목적
기존 파일의 특정 위치를 목표 지향적으로 수정한다.
### 특징
- 권한 필요
- 작은 패치/부분 수정에 적합

## 5. Write
### 목적
새 파일 생성 또는 기존 파일 전체 덮어쓰기
### 특징
- 권한 필요
- 신규 파일/전면 재작성에 적합

## 6. Agent
### 목적
subagent를 띄워 별도 컨텍스트에서 작업시킨다.
### 특징
- 무권한
- 탐색/검토/병렬 작업에 적합

## 7. Skill
### 목적
skill 기반 워크플로우를 실행한다.
### 특징
- 권한 필요
- 새 도구를 추가하는 것이 아니라 Skill 도구를 통해 기존 하네스를 사용
