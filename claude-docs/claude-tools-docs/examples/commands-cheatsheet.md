# Claude Code Tools 빠른 참조

## 탐색
- Read: 파일 읽기
- Glob: 패턴 파일 찾기
- Grep: 파일 내용 검색
- LSP: 정의/참조/타입

## 수정
- Edit: 부분 수정
- Write: 파일 생성/전체 덮어쓰기
- NotebookEdit: 노트북 셀 수정

## 실행
- Bash: 셸 명령
- PowerShell: Windows PowerShell
- WebFetch: URL fetch
- WebSearch: 웹 검색
