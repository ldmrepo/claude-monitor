# Shell / I/O / Runtime 변수
문서 ID: CC-ENV-RUNTIME
버전: v1.0.0

## 1. Bash 실행 시간
- BASH_DEFAULT_TIMEOUT_MS
- BASH_MAX_TIMEOUT_MS
- BASH_MAX_OUTPUT_LENGTH

## 2. 작업 디렉터리 및 환경
- CLAUDE_BASH_MAINTAIN_PROJECT_WORKING_DIR
- CLAUDE_ENV_FILE
- CLAUDE_CODE_SHELL
- CLAUDE_CODE_SHELL_PREFIX

## 3. 프로세스/런타임 표지
- CLAUDECODE
- CLAUDE_CODE_RESUME_INTERRUPTED_TURN
- CLAUDE_CODE_EXIT_AFTER_STOP_DELAY

## 4. 파일/검색 관련 런타임
- CLAUDE_CODE_GLOB_HIDDEN
- CLAUDE_CODE_GLOB_NO_IGNORE
- CLAUDE_CODE_GLOB_TIMEOUT_SECONDS
- USE_BUILTIN_RIPGREP

## 5. 임시 파일과 경로
- CLAUDE_CONFIG_DIR
- CLAUDE_CODE_TMPDIR

## 6. Windows/PowerShell
- CLAUDE_CODE_GIT_BASH_PATH
- CLAUDE_CODE_USE_POWERSHELL_TOOL

## 7. 운영 메모
- Bash는 작업 디렉터리는 유지되지만 export 환경변수는 명령 간 유지되지 않음
- 지속 환경은 CLAUDE_ENV_FILE 또는 SessionStart/CwdChanged/FileChanged hooks로 관리
