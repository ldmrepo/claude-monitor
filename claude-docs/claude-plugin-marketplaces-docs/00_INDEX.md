## 문서 목적
이 문서 세트는 Claude Code의 Plugin Marketplace를 실무 관점에서 이해하고, 팀 또는 조직 단위로 플러그인 카탈로그를 설계·배포·통제·운영할 수 있도록 구성한 분리형 문서다.

## 핵심 범위
- plugin marketplace의 개념과 역할
- `.claude-plugin/marketplace.json` 구조와 작성 방식
- 상대 경로, GitHub, git URL, git-subdir, npm 기반 plugin source
- 로컬/원격 호스팅 및 private repository 배포
- 팀 강제 적용, managed restriction, seed directory 운영
- validate, add/list/remove/update 등 CLI 운영 흐름
- version/channel 설계와 troubleshooting
