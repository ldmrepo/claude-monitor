# 01. MCP 개요

## 1. 정의
MCP(Model Context Protocol)는 Claude Code가 외부 도구, 데이터 소스, API, 데이터베이스와 연결되도록 하는 개방형 연동 표준이다.

## 2. Claude Code에서의 의미
Claude Code는 기본 내장 도구만으로도 파일 읽기/수정, 검색, 셸 실행을 수행할 수 있다. MCP를 추가하면 다음 범위까지 확장된다.
- 이슈 트래커 조회 및 업데이트
- 원격 API 호출
- 데이터베이스 조회
- 브라우저 자동화 도구 연결
- 디자인/협업/모니터링 시스템 연계
- 채널 기반 외부 이벤트 푸시

## 3. MCP의 핵심 가치
- Claude가 외부 시스템을 직접 다룰 수 있다.
- 프로젝트별, 사용자별, 조직 정책 기반으로 연결 범위를 통제할 수 있다.
- 로컬 stdio 서버부터 원격 HTTP 서버까지 다양한 방식으로 붙일 수 있다.
- 플러그인, Claude.ai 커넥터, 관리 정책과 연계할 수 있다.

## 4. 대표 활용 예
- GitHub PR 리뷰 및 이슈 생성
- Sentry 오류 조회
- PostgreSQL 리포트 쿼리
- Notion, Asana, Slack, Jira 연동
- 사내 전용 API/MCP 서버 연결
